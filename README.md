# TikToken CLI

A simple CLI for the TikToken calculation.

## Install

```shell
cargo install --path .
```

## Usage

```shell
A Simple tikToken wrapper CLI

Usage: tiktoken-cli [OPTIONS] --model <MODEL> --text <TEXT>

Options:
  -r, --role <ROLE>    [possible values: system, user, assistant]
  -m, --model <MODEL>
  -t, --text <TEXT>
  -h, --help           Print help
  -V, --version        Print version
```
